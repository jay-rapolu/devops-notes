# Linux Commands

### File Management

- To list all the files and directories in current path
```
ls
```
- To Check the Current path
```
pwd 
```
- To change the Directory
```
cd <path>
```
- To create a new directory
```
mkdir <dir-name>
```
- To create a empty file
```
touch <file-name>
```
- To view the file content
```
cat <file-name>
```
- To delete file
```
rm <file-name>
```
- To delete directory
```
rmdir <dir-name>
```
